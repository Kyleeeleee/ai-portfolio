智能视频分析平台

AI 大模型驱动多模态视频理解

📋 项目概述

面向智慧城市、安防监控、零售客流等场景的多模态视频分析平台。系统结合传统视觉模型（YOLOv8 目标检测）与多模态大模型（Qwen2-VL 场景理解），把"看到什么"升级为"理解正在发生什么"，并对外提供开放 API，供第三方业务系统集成。

🎯 对标：浩鲸科技 WhaleVision / 海康威视 AI 开放平台 🔗 在线Demo
注：本 demo 使用公开测试视频与模拟推理结果，用于产品能力展示，非真实业务数据。

⚡ 功能拆解

  • 目标检测：YOLOv8 实时检测，支持人 / 车 / 物体多类别标注，bbox 与置信度实时渲染
  • 场景理解：Qwen2-VL 多模态大模型输出环境 / 时段 / 密度 / 活动描述（如"傍晚商场入口人流中等"）
  • 智能问答：基于视频内容的自然语言问答（"视频中有几个人？""发生了什么事？"）
  • 分析报告：自动生成时间轴事件标记 + 人流趋势 + 类别分布图表（ECharts）
  • API 开放：Python / JS / cURL 多语言 SDK，支持第三方系统集成与二次开发

🔧 技术架构

前后端分离架构：Flask 提供 API 与控制台 UI，前端通过 AJAX 拉取分析结果并实时渲染；视频帧通过 OpenCV 解码后分别送入 YOLO 检测通道与 Qwen-VL 理解通道，结果汇总后写入时间轴事件流。

YOLOv8 Qwen-VL GroundingDINO OpenCV Flask + ECharts

📈 核心指标设计

多模态视觉 + 语言联合推理
实时目标检测 + 场景描述
开放多语言 API SDK

💡 产品洞察

视频监控行业正从"人工查看"转向"AI 理解"。中国安防摄像头超 6 亿个，AI 视频分析是一个千亿级市场。浩鲸背靠阿里 + 中兴投资，在智慧城市赛道有天然的政企客户与云资源协同优势；做"看得懂"的视频是这条赛道从硬件 PK 升级为 AI 能力 PK 的关键转折。
