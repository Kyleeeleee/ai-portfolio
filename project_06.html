📋 项目概述

内置6个Prompt模板 + 多模型并行对比评测。自动记录 token消耗、响应耗时、成本，生成可视化评测报告。

🎯 对标：快商通 产品助理 🔗 在线Demo

⚡ 核心功能

  • 6个内置Prompt模板
  • 多模型并行对比（DeepSeek/GPT/Claude）
  • Token消耗 + 耗时 + 成本分析
  • 自动生成评测对比报告
  • Prompt版本管理 + A/B测试

🔧 技术实现

Python 构建评测 Pipeline：Prompt 模板注入变量 → 并行调用多模型 API → 收集响应指标 → 自动对比分析。支持自定义评测维度和权重。

Prompt Engineering 评测框架 多模型 DeepSeek Python

💰 商业价值

Prompt 优化缺乏量化标准，本平台提供客观数据支撑，将 Prompt 迭代效率提升5倍+。
