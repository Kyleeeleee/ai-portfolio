智能客服Agent

LLM意图识别 · Tool Calling · FAQ检索 · 工单系统

📋 项目概述

基于 LLM 与 Tool Calling 的智能客服Agent。覆盖意图识别、FAQ 检索、订单查询、工单创建和智能转人工，支持多轮会话上下文管理。

🔗 在线Demo
注：本 demo 用于产品能力展示，不构成真实商业承诺。

⚡ 功能拆解

  • LLM意图识别：自动分类用户消息（FAQ咨询/订单查询/投诉/转人工）
  • FAQ智能检索：Sentence-Transformers语义匹配 + TF-IDF关键词双路召回
  • 工具调用：check_order / search_faq / create_ticket / transfer_human 四大工具
  • 智能转人工：复杂问题自动升级，附带完整对话摘要移交人工客服

🔧 技术架构

DeepSeek API 驱动意图分类与工具调用决策。Sentence-Transformers (paraphrase-multilingual-MiniLM-L12-v2) 实现FAQ语义检索，TF-IDF作为fallback。Session级别内存管理保证多轮对话连贯性。

💡 产品洞察

客服Agent的核心不是'能聊天'，而是'能办事'。Tool Calling让Agent不只是回答问题，还能真正执行查订单、建工单、转人工等操作。快商通作为企业客服SaaS厂商，Agent能力正是其产品从'在线客服'升级为'AI客服中台'的关键。
