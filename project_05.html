📋 项目概述

基于 LLM + Tool Calling 架构的智能客服系统。自动完成意图识别 → FAQ检索 → 工单创建 → 智能转人工，内置情感分析模块。

🎯 对标：快商通 🔗 在线Demo

⚡ 核心功能

  • 意图识别 + 实体提取
  • FAQ知识库检索 + 相似度匹配
  • 自动工单创建 + 分类
  • 情感分析 + 负面情绪自动升级
  • 智能转人工 + 对话摘要传递

🔧 技术实现

Agent 架构：LLM 作为大脑，Tool Calling 调度知识库检索/工单系统/情感分析等工具。对话管理模块维护状态机，支持多轮交互。

智能客服可处理80%常见问题，将人工客服工作量降低至仅处理复杂升级case，ROI显著。
