知识图谱可视化平台

实体关系挖掘与智能推理

📋 项目概述

面向国防情报、企业关系、行业研究场景的知识图谱可视化平台。系统把结构化与非结构化数据中的"实体-关系-实体"抽取出来，构建可视化图谱，并基于图算法做路径推理与自然语言问答，让用户能像搜索网页一样"搜关系"。

🔗 在线Demo
注：本 demo 图谱使用脱敏公开样例数据，用于产品能力展示，不涉密。

⚡ 功能拆解

  • 力导向图：ECharts 实时渲染实体节点 + 关系连线，支持拖拽 / 缩放 / 点击展开
  • 实体挖掘：输入名称搜索，展示属性 / 关系 / 所属知识域（国 / 企 / 装 / 人）
  • 智能推理：Dijkstra 最短路径推理（A → B 通过哪些关系关联），可视化路径
  • 自然语言问答："歼-20 和 055 型驱逐舰有什么关系？" → 图谱推理回答
  • 多知识域：国防安全 / 企业关系 / 行业知识三大预设域，一键切换

🔧 技术架构

底层 Neo4j 存储实体与关系，前端通过 Flask + ECharts 力导向图渲染；LLM 风格的实体抽取层负责从文本中识别三元组，问答层把用户问题转为 Cypher 查询并返回可读结果。

Neo4j LLM 实体抽取 Cypher Dijkstra ECharts + Flask

💡 产品洞察

Gartner 预测 2027 年 60% 的企业将使用知识图谱。从 Palantir 的国防情报到 Google 知识面板，图谱正在成为 AI 推理的下一个基础设施。渊亭在国防 AI 方向有独特的赛道壁垒——能把多源异构数据"连成关系网"的产品，是企业级 AI 决策的下一个分水岭。
