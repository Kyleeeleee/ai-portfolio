AI知识库问答助手

RAG检索增强生成 · ChromaDB向量检索

📋 项目概述

基于 LangChain + RAG 架构构建的智能知识库问答系统。用户上传文档后，系统自动解析并建立向量索引，支持自然语言问答并附带原文引用来源。

🔗 在线Demo
注：本 demo 用于产品能力展示，不构成真实商业承诺。

⚡ 功能拆解

  • 文档接入：支持 PDF / Word / TXT / Markdown 统一解析与向量化
  • 智能检索：ChromaDB 向量检索 + 关键词召回组合定位精确段落
  • 引用回答：每次回答附带原文来源链接，支持人工复核验证
  • 追问上下文：保留多轮对话历史，支持连续追问与深入探索

🔧 技术架构

ChromaDB 持久化向量存储 + ONNX MiniLM-L6-v2 本地 Embedding（384维）。langchain-text-splitters 实现智能分块，DeepSeek API 提供LLM推理。全程本地化Embedding，数据不出域。

RAG ChromaDB LangChain DeepSeek Flask

📈 核心指标

384维向量维度
500字分块大小
<2秒检索延迟

💡 产品洞察

企业知识管理的核心痛点不是'没有文档'，而是'找不到答案'。RAG方案让AI直接在已有文档中检索答案而非凭空生成，解决了大模型'幻觉'问题。深言未来正是看中了这条赛道——用AI把沉睡的企业文档变成可对话的知识资产。
