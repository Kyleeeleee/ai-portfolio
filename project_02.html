📋 项目概述

基于 LangChain + RAG 架构构建的智能知识库问答系统。用户上传文档后，系统自动解析并建立向量索引，支持自然语言问答并附带原文引用来源。

🎯 对标：深言未来 🔗 在线Demo

⚡ 核心功能

  • 支持 PDF/Word/TXT/Markdown 多格式
  • 自然语言问答 + 来源段落精准引用
  • B端企业管理后台 + C端问答界面
  • 向量检索 + 关键词混合搜索
  • 对话历史记忆 + 多轮追问

🔧 技术实现

LangChain 构建 RAG Pipeline：文档解析 → 文本分割 → Embedding → ChromaDB 向量存储 → DeepSeek 生成回答。前后端分离，RESTful API。

RAG LangChain DeepSeek Flask ChromaDB Python

💰 商业价值

企业知识管理痛点：文档散落、查找低效。本系统实现'问即所得'，将知识检索时间从平均15分钟降至30秒。
