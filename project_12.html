GenAI Agent 对话平台

Agent 编排 · 工作流引擎 · API 开放平台

📋 项目概述

面向 RingCentral / UCaaS + CCaaS 场景的企业级 AI Agent 平台。系统把"Agent 配置 / Workflow 编排 / API 开放 / 对话预览 / 平台统计"五大能力整合到一个控制台，让企业用户能像搭积木一样搭出自己业务专属的 AI 助手，并对外提供稳定的 API 与 SDK。

注：本 demo 数据使用模拟样例，用于产品能力展示。

⚡ 功能拆解

  • Agent Builder：可视化配置 Agent（名称 / 角色 / System Prompt / 工具 / 知识库 / 模型参数）
  • Workflow Designer：拖拽式工作流编排（Agent 节点 + 条件分支 + API 调用 + 循环）
  • API Playground：Python / JS / cURL SDK，请求 / 响应 JSON 预览，所见即所得
  • 对话预览：实时测试 Agent，支持打字机效果 + 流式输出
  • 平台统计：Agent 性能指标（响应时间 / token 消耗 / 成功率 / 满意度）

🔧 技术架构

基于 LangGraph + CrewAI 的多 Agent 编排框架，工具调用 + RAG 提供长期记忆与外部数据接入；Flask 提供 REST API 与前端控制台，对话层使用流式输出以模拟真实生产体验。

💡 产品洞察

2026 年是 AI Agent 元年。企业不再满足于单一 ChatBot，而是需要多 Agent 协作解决复杂业务流程。RingCentral 的 RingSense AI 正在构建 UCaaS + Contact Center + AI 的三角飞轮——这个平台是下一代企业 AI 的基础设施，让每一家公司都能拥有自己的"数字员工团队"。
