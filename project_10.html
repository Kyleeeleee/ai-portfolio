AIoT 智能家居控制台

小米生态链 AIoT 设备统一管理平台

📋 项目概述

面向小米生态链 / 米家用户与智能家居集成商的 AIoT 控制台。系统把分散在不同房间、不同协议的 IoT 设备统一汇聚到一个控制台，提供"3D 户型可视化 + AI 场景联动 + 能耗分析 + 语音控制"四件套，让用户能在一个界面里看到、理解和自动化管理整个家。

🎯 对标：大白科技 / 米家 / Aqara 全屋智能方案 🔗 简历项目区入口
注：本 demo 设备状态由模拟数据驱动，用于产品能力展示。

⚡ 功能拆解

  • 全屋设备管理：10 类设备统一控制（灯 / 空调 / 门锁 / 摄像头 / 窗帘 / 扫地机 等）
  • 3D 房间可视化：SVG 绘制 9 区域户型图，设备位置实时标注 + 点击设备查看详情
  • AI 场景联动：5 个预设场景（回家/离家/观影/睡眠/起床）+ 3 条 AI 规则引擎（PM2.5 / 温度 / 用电高峰自动触发）
  • 能耗分析：日 / 周 / 月用电趋势 + 能效占比 + AI 节能建议（基于历史模式）
  • 语音控制：自然语言指令识别，模拟 320ms ASR 延迟，提升交互真实感

🔧 技术架构

设备层用 MQTT 协议与米家 SDK 通信，事件进入规则引擎后判断是否触发预设场景；前端用 Flask + SVG 渲染 3D 户型，ECharts 绘制能耗趋势；语音模块模拟 320ms ASR 延迟以体现真实产品体验。

💡 产品洞察

全球智能家居市场 2030 年将达 $500B。小米 IoT 设备连接数超 6 亿，AIoT 的核心不是硬件而是"场景联动"——让不同品牌的设备协同工作。大白作为小米生态链企业，场景定义能力是核心竞争力；做"会主动思考的家"是产品差异化的关键。
