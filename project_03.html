📋 项目概述

基于大语言模型的游戏NPC智能对话系统，预设3个角色（导师/跟班/商人），具备上下文记忆、情感状态追踪和结构化 JSON API 输出能力。

🎯 对标：4399游戏 🔗 在线Demo

⚡ 核心功能

  • 3种NPC角色模板（导师/跟班/商人）
  • 上下文记忆 + 多轮对话连贯性
  • 情感状态追踪（友好/中立/敌对）
  • JSON API 输出，便于游戏引擎集成
  • 角色人设固化 + Prompt 模板管理

🔧 技术实现

核心采用 Prompt 链式编排：角色设定 → 情感判断 → 记忆检索 → 对话生成。记忆模块基于滑动窗口 + 关键信息摘要，API 层提供标准化 JSON 输出。

💰 商业价值

传统游戏NPC对话需大量脚本编写，AI方案将对话内容制作效率提升10倍+，且对话质量更自然、更具沉浸感。
